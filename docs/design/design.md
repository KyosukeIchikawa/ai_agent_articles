## サイト構造設計

### 全体構成

```

├── index.js (トップページ/概要)
├── background/ (背景)
│   └── index.js
├── related-work/ (関連研究)
│   └── index.js

│   └── index.js

│   └── index.js

│   └── index.js
```

## UI/UX設計

### ナビゲーション設計
- 固定ヘッダーにグローバルナビゲーションを配置
- 各セクションへのリンクを提供
- 現在位置がわかるよう視覚的に強調
- モバイル表示時はハンバーガーメニューに切り替え
- 左サイドバーに詳細な目次を表示
- 右サイドバーに現在のセクション内の小見出しリンクを表示

### ページレイアウト
- ヘッダー: サイトタイトル、ナビゲーション
- メインコンテンツ: 論文解説、図表、数式
- 左サイドバー: グローバル目次
- 右サイドバー: 現在セクション内の目次
- フッター: 著作権表示、連絡先

### レスポンシブデザイン
- PC (1200px以上): 3カラムレイアウト（左サイドバー + メインコンテンツ + 右サイドバー）
- タブレット (768px～1199px): 2カラムレイアウト（メインコンテンツ + 右サイドバー）
- モバイル (767px以下): 1カラムレイアウト（メインコンテンツのみ）

### カラースキーム
- プライマリカラー: #3498db (青系) - ロボティクス・AIを表現
- セカンダリカラー: #2ecc71 (緑系) - 好奇心・探索を表現
- アクセントカラー: #e74c3c (赤系) - 重要概念の強調に使用
- 背景色: #f5f5f5 (明るいグレー) - 読みやすさを確保
- テキスト色: #333333 (濃いグレー) - コントラストを確保

### タイポグラフィ
- 見出し: Roboto (sans-serif)
- 本文: Noto Sans JP (日本語対応フォント)
- コード: Source Code Pro (monospace)
- 数式: MathJax標準フォント

## 機能設計

### コンテンツ表示機能
- セクション別ページ表示
- 数式の動的レンダリング (KaTeX使用)
- インタラクティブな図表表示
- コードスニペット表示
- 用語集ポップアップ（専門用語にマウスオーバーで説明表示）

### ナビゲーション機能
- スムーズスクロール
- ページ内リンク（アンカーリンク）
- パンくずリスト
- 前後のセクションへのナビゲーション
- サイドバー目次の自動更新（現在位置を表示）

### インタラクティブ機能
- 図表のズーム表示
- アルゴリズム実行の簡易デモ（ICMと記号的計画の統合）
- ダークモードの切り替え

- GitHub Pagesへのホスティング
- カスタムドメイン設定（必要に応じて）

## ページ別設計

### トップページ (index.js)
- 論文タイトル（日英表記）
- 著者情報とICRA 2025情報
- 論文の概要（アブストラクト）の日本語訳
- 主要貢献のハイライト（3-4項目のカード形式）
- 研究の位置づけの図解（ロボティクス・AI分野における位置づけ）
- 各セクションへのクイックリンク

### 背景ページ (background/index.js)
- オープンワールド環境の課題
  - 不確実性と動的変化への対応問題
  - 図解: 動的環境の例示
- 従来のTAMPアプローチの限界
  - 計画修正の非効率性
  - データ効率の問題
  - 図解: 従来手法の問題点

### 提案手法ページ (method/index.js)
- システムアーキテクチャ概要
  - 図解: システム全体構成図
- ICMモジュールの詳細
  - 内発的好奇心の仕組み
  - 探索駆動の学習メカニズム
  - 数式: ICMの定式化
- 記号的計画モジュール
  - オペレータの発見メカニズム
  - 想像空間での計画生成
  - 図解: 想像空間の表現方法
- 報酬機械（Reward Machine）
  - 生成プロセス
  - 数式: 報酬機械の定式化
- インタラクティブデモ: 簡略化された例での動作説明

### 実験と結果ページ
- 実験設定の詳細
  - ロボット操作タスクの説明
  - 新規性注入のプロセス
  - 図解: 実験環境
- 比較手法の説明
- 定量的結果
  - 図表: 性能比較グラフ
  - 表: 収束速度の比較
- 定性的分析
  - ケーススタディ
  - 図解: 特徴的な適応例
